# Vue 電商作品 - The Florist 花藝電商

## 設計概念

線上花藝工作室，販售花藝以及相關周邊
色系：咖啡色、米白
圖片來源： Unsplash[https://unsplash.com/]

- Vue.js
- Vue CLI
- Vue Router
- Vue Axios
- Vee Validate
- Vue Loading Overlay
- Mitt
- Bootstrap
- CKEditor
- Swiper.js
- ESlint Prettier
- Material icons
